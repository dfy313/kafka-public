# 📺 Kafka – Section 1a

In this first section, we’ll introduce the E-Commerce App that we’ll be using throughout the course and explore how it communicates with Kafka. You’ll see how each microservice (Order, Payment, Notification) acts as a producer or consumer, exchanging events through Kafka topics — all before we start building our own “Kafkaesque” cluster from scratch. This demo provides intuition for how event-driven systems behave in practice and how Kafka acts as the central message backbone.
